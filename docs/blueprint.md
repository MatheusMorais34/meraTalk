## Style Guidelines:

- Primary color: Vibrant red (#E63946) for energy and call to action.
- Background color: Soft, desaturated off-white (#F8F4F4) for a modern look.
- Accent color: Slightly darker red (#C42730) for highlighting key elements. Should have significantly different brightness/saturation to create contrast.
- Font: 'Inter' for both headings and body, to keep a contemporary feel and maintain easy readability
- Simple, flat icons in red or dark gray to represent different call features.
- Clean, sectioned layout with plenty of whitespace to ensure readability.
- Subtle animations on button hovers and when loading new data.
